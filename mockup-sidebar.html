Ting! — Mockup nâng cấp Sidebar

So sánh 4 phương án cạnh nhau. Rê chuột vào các mục để xem hiệu ứng hover. Mục "Nhóm" đang ở trạng thái active giống ảnh của bạn.
Hiện tại
Bản đang chạy. Icon đồng màu, badge tím đồng loạt, active chỉ có viền trái.
A · Tinh chỉnh nhẹ
Icon có màu riêng theo nhóm, badge phân loại ngữ nghĩa (Thùng rác đỏ). Thay đổi tối thiểu, an toàn.
B · Floating Pill ★ đề xuất
Active kiểu viên thuốc nổi + glow, thanh sáng bo tròn, icon micro-animation, badge glass, nút Add shimmer, footer có user profile.
C · Glass / Icon-chip
Nền kính mờ, mỗi icon trong ô bo tròn kiểu iOS, active làm sáng ô icon + chấm phát sáng.
D · Neon / Cyber
Phong cách gaming: viền phát sáng neon, icon active glow, badge dạng viền rỗng. Hợp tông tối, cá tính mạnh.
E · Light Mode
Sidebar nền sáng sạch sẽ, chữ đậm dễ đọc. Phù hợp nếu bạn muốn thêm chế độ sáng cho app.
F · Aurora Header
Vùng logo có dải sáng aurora chuyển động nhẹ, item active là thẻ nổi mềm + thanh gradient trái.
G · Full Gradient (Discord-style)
Item active phủ nguyên nền gradient tím nổi bật, tương phản cao, nhìn phát là biết đang ở đâu.
